centos如何进入（安装）图形界面

centos默认不会安装图形界面所以要手动安装

1）获取root权限

login: root
Password:

2）安装X Window System

yum groupinstall “X Window System”

输入y继续安装，安装完成最后显示complete！

3）安装图形界面软件

yum groupinstall “GNONE Desktop”

输入y继续安装，安装完成最后显示complete！

4）安装完成输入startx打开图形界面
